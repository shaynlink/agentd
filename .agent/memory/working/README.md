# Memory — Working Memory

This directory holds temporary, session-scoped context.

## Usage

Files here represent the current agent's working context for an active session.
They can be created and deleted per session.

## Examples

- `current-task.md` — Active task description and constraints
- `session-assumptions.md` — Assumptions made during this session
- `pending-questions.md` — Unresolved questions to surface to the user
