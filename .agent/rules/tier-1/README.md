# Tier 1 — Technical Core Skills

Place project-specific technical skills here. These are loaded for technical tasks.

## Naming Convention

Files should follow: `NN-skill-name.md` (two-digit prefix, kebab-case)

## Examples

| File | Purpose |
|---|---|
| `01-language-mastery.md` | TypeScript / Python / Go standards |
| `02-clean-architecture.md` | Code structure and SOLID principles |
| `03-security.md` | Security standards and compliance |
| `04-database.md` | DB query patterns and indexing |
| `05-api-standards.md` | Rest / GraphQL / gRPC conventions |
| `06-testing.md` | Testing pyramid and quality gates |
